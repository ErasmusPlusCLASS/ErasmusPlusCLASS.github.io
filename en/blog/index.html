Совещание по Skype

written by KazNU on 2018-09-11

11 сентября 2018 года участники проекта из Костанайского государственного университета имени А.Байтурсынова, Евразийского национального университета им. Л.Н. Гумилева и Казахского национального университета им. аль-Фараби провели совещание по Skype. Во время совещания обсуждались текущие работы по проекту и действия участников провекта по распространению информации о проекте среди студентов и магистрантов.

Мониторинг проекта

written by KazNU on 2018-06-13

03 мая 2018 года в Костанайском государственном университете имени А.Байтурсынова прошёл мониторинг проекта Развитие междисциплинарной магистерской программы по вычислительной лингвистике в университетах Центральной Азии - CLASS 585845-ЕРР-1-2017-1-ES-EPPKA2-CBHE-JP.

Во время мониторинга были представлены сведения о работах, выполненных командой проекта в казахстанских ВУЗах. Представители Национального офиса Эразмус+ в Казахстане сообщили большое количество полезной информации, касающейся проекта, высказали замечания и дали рекомендации по дальнейшему развитию проекта.

Организационное совещание

written by KazNU on 2018-06-12

19-20 февраля 2018 года прошло организационное совещание в рамках проекта Erasmus+ CLASS «Разработка междисциплинарной магистерской программы по вычислительной лингвистике в университетах Центральной Азии».

Место проведения: Университет Сантьяго де Компостела (University of Santiago de Compostela), Испания.

На совещании были обсуждены план координирования проекта, утвержден предварительный план действий по проекту, были утверждены ответственные по отдельным направлениям деятельности и уточнены сроки выполнения работ. Были представлены доклады о положениях Болонского процесса и особенностях технологии смешанного обучения (blended learning). Представители TEIATH (Technologiko Ekpaideftiko Idryma Athinas) выступили с презентацией о планах по созданию веб-сайта проекта и трудностях его поддержки.
